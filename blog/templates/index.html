Some Web App Info

    What is a client?
  • A computer hardware device or software that accesses a information and applications from a server
  • For example, a desktop computer or a mobile device
    What is a server?
  • A computer that provides data to other computers
  • For example, a web server, mail server, or file server
    What is an HTTP request?
  • An HTTP request consists of a URL, request method, a header, and sometimes a body
  • Basically it outlines what the client wants and what action they want to take
  • Common reuqest methods include GET, POST, PUT/PATCH, and DELETE
  • These request methods allow you to fetch, create, update, and delete data
    What is an HTTP response?
  • An HTTP response consists of a status code, a header, and a body of data
  • Basically it is the outcome of the action taken by the server once a request is received from a client
  • There are various response codes that can provide information about the request, redirect, or provide an error message
  • Learn about HTTP status codes here.
    What is DJANGO and why do we use it?
  • DJANGO is an open source web application framework in Python
  • DJANGO allows users to build secure and maintainable websites without having to rewrite all of the code from scratch
